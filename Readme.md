# 🚀 AI Resume & Portfolio Builder

An intelligent, blazing-fast web application that generates fully structured, ATS-optimized resumes in Markdown format using the Groq API and Streamlit. 

live project link :- [click here](https://resumebuilder777.streamlit.app/)

## ✨ Features
* **Instant Generation:** Powered by Groq's lightning-fast inference models (Llama 3.1).
* **ATS-Optimized:** Structures your experience and skills specifically for Applicant Tracking Systems.
* **Markdown Export:** Generates clean, ready-to-use Markdown text with a one-click download feature.
* **Clean UI:** Built with an intuitive, multi-column Streamlit interface.

## 🛠️ Tech Stack
* **Frontend:** Streamlit
* **Backend/Logic:** Python
